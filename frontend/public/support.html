‚ùì Frequently Asked Questions

Why am I not receiving notifications?

Common solutions:

  • Check browser notification permissions (Settings ‚Üí Notifications)
  • Verify your email address in NotifyPro settings
  • Make sure business hours are configured correctly
  • Disable Do Not Disturb mode on your device
  • Test with the "Send Test Notification" button

How do I set up Slack notifications?

Follow these steps:

  1. Go to https://api.slack.com/apps
  2. Create a new app or select an existing one
  3. Enable "Incoming Webhooks"
  4. Add webhook to your workspace
  5. Copy the webhook URL and paste it in NotifyPro settings

Can I customize which conversations trigger notifications?

Yes! NotifyPro offers several filtering options:

  • Business Hours: Only receive notifications during work hours
  • Priority Keywords: Get special alerts for important messages
  • New Conversations Only: Only notify for new chats
  • First Message Only: Just the first message in a thread

üîß Quick Troubleshooting

Browser Notifications Not Working?

Try these steps:
  1. Check browser settings: chrome://settings/content/notifications
  2. Ensure NotifyPro is allowed in browser permissions
  3. Disable browser Do Not Disturb mode
  4. Re-enable push notifications in NotifyPro settings
  5. Clear browser cache and try again

Email Notifications Missing?

Check:
  1. Verify your email address in settings
  2. Check spam/junk folder
  3. Add support@vaultsuite.store to contacts
  4. Make sure email notifications are enabled
  5. Test with "Send Test Notification"

Slack Not Receiving Notifications?

Verify:
  1. Webhook URL is correct
  2. Webhook is active in Slack (not revoked)
  3. Slack app has proper permissions
  4. Test webhook directly with curl
  5. Check Slack channel settings
